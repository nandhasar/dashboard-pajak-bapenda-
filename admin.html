Konfigurasi API

Petunjuk Setup

Metode 1 - OAuth2 (Recommended):

  1. Buka Google Cloud Console
  2. Aktifkan Google Sheets API
  3. Buat OAuth 2.0 Client ID
  4. Download JSON credentials
  5. Gunakan tombol "Masuk dengan Google"

Metode 2 - API Key:

  1. Buka Google Cloud Console
  2. Aktifkan Google Sheets API
  3. Buat API Key (dengan batasan aplikasi)
  4. Bagikan spreadsheet dengan "Anyone with link"
  5. Masukkan API Key di field di atas
